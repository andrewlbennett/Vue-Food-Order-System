# Vue-Food-Order-System
This is a vue front-end for a food ordering system.

### Mockup
![alt text](https://raw.githubusercontent.com/meadio/Vue-Food-Order-System/master/mockup.png)

### Knowledge Required
* HTML
* CSS/SCSS
* Javascript
* [Vue](https://vuejs.org/)
* Basic Command Line

### Prerequisites
* node 8.0+
* npm 5.0+

### Installation
```
# install
npm install
```

### Development
```
# run
npm run dev
```
